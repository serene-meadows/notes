in go, 
  generating random numbers invluves using the math Rand package, which provides functions for genreating pseuo random and crytographically secure random numbers

  a seed is a starting point for genreating a sequence of random numbers 
  in go, the default psudo random number generator provided by the math Rand uses a seed to initialize the sequence 
  rand uses a deterministic algorithm to generate psudo random numbers based on the seed 


pseudo random numbers 
  they are generad by algorithms, they are not trully random numbers
  these are numbers that appeare random but are actually generated by algorithms based on an initial seed value

  the pseudonym is PRNG (Psudo Random Number Generator)

to generate tru random numbers 
  you need to generate them from physicall processes, like radioactive decay or thermal noise


seed 
  a seed is an inital value used by the pseudo random number generator to start egnerating a sequence of numbers that the generator will produce
  the purpose of seed is reproducibility
  by setting the same seed ,you can generate the same sequence of random numbers each time you run your program
    which can be used for debugging and testing 

 thats why traditionally, we use time as the seed, so a different number is generated each time the program is executed


example - Auto Seed
  ````go
    import (
      "fmt"
      "math/rand"
    )

    func main() {
      fmt.Println(rand.Intn(101))
        // generates a random number between 0 to 101
          // range is [0. n)
        // this is auto seeded
    }
  ````


example - Fixed Seed
  ````go
    import (
      "fmt"
      "math/rand"
    )

    func main() {
      val := rand.New(rand.NewSource(42))
      // to use time 
        // rand.New(rand.NewSource(time.Now().Unix())) // .Unix() to convert to int

      fmt.Println(val.Intn(101))
      
  ````
