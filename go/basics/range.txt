the range keyword provides a convenient way to iterate over various data structures like arrays,
slices, strings, maps, channels without needing to deal with indices or iterators


range keyword operates on a copy of the data structure it iterates over
therefore, modifying index or value inside the loop doesn't affect the original data structure
  i.e. if we modify any value inside range loop, modification will not affect the original value


for arrays, slices, strings
    range iterates in order from the first element to the last

for maps
    range iterates over key, value pairs in an unspecified order

for channels
    range iterates until the channel is closed


if we do not intend to use one of the values received remem to use blank identifier `_`


strings
  message := "hello world"

  for i, v := range message {
    fmt.Println(i, v)
      // i -> index
      // v -> unicode value of character

    // to get actual character instead of unicode (in other words to get runes)
      // characters are called runes in go
    fmt.Printf("Index: %d, Rune: %c", i, v)
  }
