_
  blank identifier
  
  used when we do not want to use a value that is being returned
  needed because go gives an error if a variable is declared but not used

  example 
    ```go
      func someFunction() (int, int) {
        return 1,2
      }

      // no error, both values captured, both values used
      a, b = someFunction()
      fmt.Println(a)
      fmt.Println(b)

      // no error, both value captured, one value used, one value -> blank identifier 
      a, _ = someFunction()
      fmt.Println(a)

      // error, both values captured, one value used
      a, b = someFunction()
      fmt.Println(a)

      // error, one value captured 
      a = someFunction()
      fmt.Println()
    ```
