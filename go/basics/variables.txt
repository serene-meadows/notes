syntax for declaring variables
  ex
    // uninitialzed variable
    var age int

    // initialized variable
      // type of variable is optional if initialzed, its inferred
    var name = "vikas"
    var name_2 string = "goudar"

    // wierd symbol method
      // type and `var` inferred
        // var keyword cannot be used
        // type cannot be written
      // it can only be used within a function
        // example cannot be used in packge("global") scope
   count := 0


variables declared without explicit initialization are zero-initialized

variables have block scope
  blocks can be
    if, for, switch, select, function, explicit {} blocks, case clause, package

  variables can only be accessed within their scopes because
    variables lifetime is within their scope, 
      they are created when their scope is entered and destroyed when their scope is exited

package variables
  "global variables" are actually package-level variables
  they are visible to all files in the same package

  cannot be declared using := notation (:= can only be used within functions)
  have to use var declaration
