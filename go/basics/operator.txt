Logical Operator
  || (logical or)
    condiition_1 || condition_2
      if condiition_1 is true, the compiler doesn't evaluate condition_2


  && (logical and)
    condition_1 && condition_2
      if condition_1 is false, the compiler doesn't evaluate condition_2


Bitwise Operator
  ^
    unary -> not (2s complement)
    binary -> xor

  &^
    bit clear operator
    (and not)
