default/ zero values 
  int -> 0
  floating point numbers -> 0
  complex numbers -> 0 + 0i

  strings -> ""

  constants -> have to be initialized on declearation,
               thus no concept of zero-initialized


  arrays -> each element gets its type's zero value

  structs -> its fields are zero-initialized

  pointers -> nil

  maps -> nil

  slices -> nil

  functions -> nil

  channels -> nil

  json -> when unmarshalling into a struct, 
            its fields are zero initialized 

  error interface -> nil
