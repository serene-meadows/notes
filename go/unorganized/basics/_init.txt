init function
  is a special function that can be declared in any package
  its used to perform initialization tasks for the package before it is used

  go executes the init function automatically when the package is initialized
  this happens before the `main` function is executed
  the init fucntion is executed once per package, even if the package is imported multiple times
    int multiple files

  order of exeution
    At run time, a package will be loaded after all its dependency packages.
    Each package is loaded once and only once 

    All init functions in all involved packages in a program will be invoked sequentially.
    An init function in an importing package will be invoked after all the init functions declared in the dependency packages 
      of the importing package 
    All init functions will be invoked before invoking the main entry function.

    The invocation order of the init functions in the same source file is from top to bottom
    Go specification recommends, but doesn't require, to invoke the init functions in different source files of the same package by the alphabetical order of filenames of their containing source files.
      but in general it's not a good idea to have dependency relations between two init functions in two seperate source files 

  
    All package-level variables declared in a package are initialized before any init function declared in the same package is invoked.

its used
  to initialize variables
  setup operations
  registering components or configurations
  intializing state required for the package to function

  one use case is
    open database connections
    perform schema migrations needed by the package


declaration
  the init function hsa no parameters and no return values

  example
    ```go
      package main

      import "fmt"

      func init() {
        fmt.Println("Initializing package")
      }

      func main() {
        fmt.Println("Inside the main function")
      }
    ```

    // output
      Initializing package
      Inside the main function


  example - multiple init functions
    ```go
      package main

      import "fmt"

      func init() {
        fmt.Println("Initializing package - 1")
      }
      func init() {
        fmt.Println("Initializing package - 2")
      }
      func init() {
        fmt.Println("Initializing package - 3")
      }

      func main() {
        fmt.Println("Inside main function")
      }
    ```
    // output
      Initializing package - 1
      Initializing package - 2
      Initializing package - 3
      Inside main function
