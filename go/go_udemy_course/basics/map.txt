maps are a builtin datastructure that associate keys with values
  each key must be unique within the map
    map keys must be comparable
      a type is comparable if the logical comparison operators (== and != can be applied to it)
      its needed to test if a key already exists

maps are unordered collections of key value pairs

declaration
  var mapName map[keyType]valueType // zero initialized to nil

  var mapName map[keyType]valueType = map[keyType]valueType {}
  var mapName = map[keyType]valueType {}


  mapName := map[keyType]valueType{
    key1: value1,
    key2: value2,
  }

  // using make
  mapName := make(map[keyType]valueType)


  // nested maps
  outerMap := make(map[string]map[string]string) 
  innerMap := make(map[string]string)
    outerMap["map1"] = innerMap


// set key, value
mapName[key] = value

// get value
mapName[key]
  // if a key doesn't exist, zero-value of valueType is returned

  // there is another optional return-value that is returned along with mapName[key] apart from the value
  ```golang
    // without using optional return-value
    value := mapName["string-key"]
      // gets value, zero-value of valueType if key doesn't exist

    // with using optional return-value
    value, ok := mapName["string-key"]
      // ok
        // true -> if key exists
        // false -> if key doesn't exist

  ```
