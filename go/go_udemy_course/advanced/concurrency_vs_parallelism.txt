concurrency 
  the ability of a system to handle multiple tasks simultaenously 
  it involves managing multiple tasks that are in progress at the same time but not necessarily executed at the same time

  multiple tasks are in progress at the same time
  the system switches between tasks, giving the illusions that they run at the same time

parallelism
  the simultaenous execution of multiple tasks, typically using multiple processors or cores, to improve performance 
  by running operations at the same time  

  multiple tasks actually run at te same time 
  requires multiple cores

```go
  func heavyTask(id int, wg *sync.WaitGroup) {
    defer wg.Done()
    fmt.Printf("Task %d is starting", id)

    for range 100_000_000 {
    }
    fmt.Println(time.Now())

    fmt.Printf("Task %d is finished \n", id)
  }

  func main() {
    numThreads := 4

    runtime.GOMAXPROCS(numThreads)
    // sets the maximum number of CPUs that we can use simultaenously

    var wg sync.WaitGroup
    for i := range numThreads {
      wg.Add(1)
      go heavyTask(i, &wg)
    }

    wg.Wait()
  }
```
