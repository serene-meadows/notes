-- interface value
--
  Interface value = value whose type is an interface 
  it can hold ("box") a non-interface value
  
 
  -- terminology
  --
    dynamic value -> the actual value stored
    dynamic type  -> the type of that value

  
  -- dynamic type and dynamic value
  --
    var i interface{} = 10

    dynamic value = 10
    dynamic type = int

    ```go
      package main

      import "fmt"

      func main() {

        var i interface{}

        i = 10
        fmt.Printf("%T %v\n", i, i) // int 10

        i = "hello"
        fmt.Printf("%T %v\n", i, i) // string hello
      }

    ```





-- zero interface value
--
  an interface holding nothing = zero interface value

  -- properties of zero interface value
  --
    dynamic type -> nil
    dynamic value -> nil

  example
    ```go
      package main 

      import "fmt"

      func main() {
        var i interface{}

        fmt.Println(i == nil) // true
      }

    ```


  -- when interface is = to nil
  --
    this is a common gotcha

    an interface is nil only if both dynamic type and dynamic value are nil
    when you declare `var i interface{}` -> both fields are empty so `i == nil` is true

    but when you do, for example
      ```go
        package main

        import "fmt"

        func main() {
          var p *int = nil
          var i interface{} = p
          fmt.Println(i == nil) // false
        }

      ```
        // i has:
        //   dynamic type  = *int
        //   dynamic value = nil
        //   -> interface is NOT nil because it has a type

    -- pseudo explanation
    --
      think of interface as a struct with two fields
        pointer to dynamic type
        pointer to dynamic value

      unless both are nil, the interface carries some value and thus isn't = nil





-- interface implementation
--
  there is no explicit 'implement' keyword
  a type implements an interface if 
    method set of interface is a subset of method set of type 

  example
    ```go
      package main

      import "fmt"

      type Speaker interface {
        Speak()
      }

      type Dog struct{}

      func (Dog) Speak() {
        fmt.Println("bark")
      }

      func main() {
        var s Speaker = Dog{} // Dog implements Speaker
        s.Speak()
      }

    ```

  -- an interface can implmenet another interface 
  --
    ```go
      type A interface {
        F()
      }

      type B interface {
        F()
        G()
      } // B implements A
      
    ```





-- concrete value and concrete type of a value
--
  For a (typed) non-interface value, its concrete value is itself and its concrete type is the type of the value.
  A zero interface value has neither concrete type nor concrete value.
  For a non-zero interface value, its concrete value is its dynamic value and its concrete type is its dynamic type.
