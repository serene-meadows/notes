-- Types Which Support or Don't Support Comparisons
--
  Go doesn't support comparisons (with the == and != operators) for values of the following types:
    Slices: You cannot compare two slices (e.g., s1 == s2).
    Maps: You cannot compare two maps (e.g., m1 == m2).
    Functions: You cannot compare two function values (e.g., f1 == f2).
    -- recurses
    --
      Structs: A struct is incomparable if any of its fields are incomparable.
      Arrays: An array is incomparable if its element type is incomparable.
      Example: An array of integers [5]int is comparable, but an array of slices [5][]int is not.

    -- one exception
    --
      even though you cannot compare the incomparable types, you can compare them to the literal 'nil'
      ```go
        var s []int
        fmt.Println(s == nil) // This is LEGAL (true)

        s2 := []int{1, 2}
        // fmt.Println(s == s2) // ILLEGAL: invalid operation

      ```
  
  All other types are called comparable types.

  The key type of any map type must be a comparable type.
