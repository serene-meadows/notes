-- identifier
--
  An identifier is a token which must be composed of Unicode letters, Unicode digits and _ (underscore)
  it starts with either an Unicode letter or _  

  keywords can not be used as identifiers.

  identifier _ is a special identifier, it is called blank identifier.

  An identifier starting with an Unicode upper case letter is called an exported identifier.
  The word exported can be interpreted as public in many other languages. 

  The identifiers which don't start with an Unicode upper case letter are called non-exported identifiers.
  The word non-exported can be interpreted as private in many other languages
  eastern characters are viewed as non-exported letters
